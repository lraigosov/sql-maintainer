# Nivel 1: Fundamentos SQL

Este módulo establece las bases del lenguaje SQL y el pensamiento relacional.

## Objetivos de Aprendizaje
- Comprender el modelo relacional (Tablas, PKs, FKs).
- Dominar la sintaxis básica: `SELECT`, `FROM`, `WHERE`, `ORDER BY`.
- Aprender a combinar tablas con `JOIN`.
- Realizar agregaciones básicas (`GROUP BY`).

## Contenido del Módulo

| Notebook | Tema | Conceptos Clave |
|----------|------|-----------------|
| `01_introduccion_relacional.ipynb` | Introducción | Modelo dimensional, PK/FK |
| `02_select_basico.ipynb` | SELECT Básico | Selección de columnas, alias |
| `03_funciones_tipos.ipynb` | Funciones | Strings, Fechas, Conversiones |
| `04_filtrado_avanzado.ipynb` | Filtrado | `IN`, `BETWEEN`, `LIKE`, `NULL` |
| `05_joins.ipynb` | Joins | `INNER`, `LEFT`, `RIGHT`, `FULL` |
| `06_agregaciones_kpis.ipynb` | Agregaciones | `SUM`, `COUNT`, `AVG`, `GROUP BY` |
| `07_consultas_multitabla.ipynb` | Multitabla | Joins complejos, lógica de negocio |
| `08_calidad_datos_basica.ipynb` | Calidad | Detección de duplicados y nulos |
| `09_proyecto_cierre_junior.ipynb` | **Proyecto** | Análisis completo de ventas |
