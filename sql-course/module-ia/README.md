# Módulo 4: IA Aplicada a SQL

Este módulo explora cómo utilizar herramientas de Inteligencia Artificial Generativa (LLMs) para potenciar el trabajo del Analista y DBA.

## Objetivos de Aprendizaje
- Aprender a escribir **prompts efectivos** para generación de código SQL.
- Utilizar IA para **explicar** y documentar consultas complejas.
- Técnicas de **refactoring** y optimización asistida.
- Generación de **datos sintéticos** para pruebas.
- Validar resultados y detectar alucinaciones.

## Contenido del Módulo

| Notebook | Tema | Descripción |
|----------|------|-------------|
| `01_prompting_efectivo.ipynb` | Prompt Engineering | Contexto, restricciones y formato. |
| `02_explicacion_sql_con_ia.ipynb` | Explicación | De código a lenguaje natural. |
| `03_refactor_sql_asistido.ipynb` | Refactoring | Mejorar legibilidad y performance. |
| `04_validacion_resultados_con_ia.ipynb` | Validación | Detectar errores lógicos y de sintaxis. |
| `05_generacion_datos_sinteticos.ipynb` | Datos de Prueba | Crear datasets realistas con Faker/LLMs. |
| `06_revision_ai_pair_programming.ipynb` | Review | Simulación de Code Review con IA. |
| `07_etica_y_gobernanza_ia.ipynb` | Ética | Privacidad, seguridad y límites. |

## Prerrequisitos
- Conocimientos sólidos de SQL (Niveles 1-3 recomendados).
- Acceso a un asistente de IA (ChatGPT, Claude, Copilot, etc.).
