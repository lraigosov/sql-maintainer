# Laboratorios Prácticos

Colección de ejercicios prácticos integradores para reforzar habilidades específicas transversales.

## Objetivos
- Aplicar conocimientos en escenarios de negocio simulados.
- Resolver problemas de punta a punta (ETL -> Análisis -> Visualización).
- Auditoría y calidad de datos.

## Laboratorios Disponibles

| Notebook | Nivel | Descripción |
|----------|-------|-------------|
| `01_dashboard_ventas_basico.ipynb` | Junior | Consultas para alimentar un dashboard de ventas mensual. |
| `02_auditoria_calidad_datos.ipynb` | Intermedio | Detectar inconsistencias, nulos y duplicados. |
| `03_optimizar_consultas_practico.ipynb` | Senior | Ejercicios de tuning sobre consultas lentas preparadas. |
| `04_mini_etl_fuentes.ipynb` | Intermedio | Simulación de carga incremental y transformación de datos. |

## Cómo usar
Estos laboratorios pueden realizarse en paralelo a los niveles principales para reforzar conceptos.
