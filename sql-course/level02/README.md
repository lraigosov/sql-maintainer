# Nivel 2: Analítica Intermedia

Este módulo se enfoca en herramientas analíticas avanzadas para resolver preguntas de negocio complejas.

## Objetivos de Aprendizaje
- Dominar las Funciones de Ventana (`OVER`, `PARTITION BY`).
- Utilizar CTEs (Common Table Expressions) para código limpio.
- Entender subconsultas y su impacto.
- Introducción al modelado dimensional.

## Contenido del Módulo

| Notebook | Tema | Conceptos Clave |
|----------|------|-----------------|
| `01_fundamentos_window_functions.ipynb` | Window Functions | `RANK`, `ROW_NUMBER`, `OVER` |
| `02_ctes_recursivas.ipynb` | CTEs | `WITH`, Recursividad básica |
| `03_subconsultas_avanzadas.ipynb` | Subconsultas | `EXISTS`, Correlacionadas |
| `04_modelado_dimensional_basico.ipynb` | Modelado | Esquema Estrella, Granularidad |
| `05_kpis_avanzados.ipynb` | KPIs | YoY, MoM, Running Totals |
| `06_segmentacion_basica.ipynb` | Segmentación | RFM (Recencia, Frecuencia, Monto) |
| `07_optimizar_consultas_basico.ipynb` | Optimización I | SARGability, evitar `SELECT *` |
| `08_control_version_datos.ipynb` | Versionado | SCD Tipo 1 y 2 (Teoría) |
| `09_integracion_fuentes.ipynb` | Integración | `UNION`, `MERGE` (básico) |
| `10_proyecto_intermedio.ipynb` | **Proyecto** | Dashboard analítico complejo |

## Prerrequisitos
- Completar el Nivel 1.
- Entender bien las relaciones entre tablas (`JOIN`).
