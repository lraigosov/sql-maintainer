# Evaluación y Certificación

Recursos para la evaluación de competencias y rúbricas de proyectos finales.

## Contenido

| Archivo | Nivel | Descripción |
|---------|-------|-------------|
| `01_rubrica_junior.ipynb` | Junior | Criterios de evaluación para el Proyecto 1 (Fundamentos). |
| `02_rubrica_intermedio.ipynb` | Intermedio | Criterios para el Proyecto 2 (Analítica). |
| `03_rubrica_senior.ipynb` | Senior | Criterios para el Proyecto 3 (Performance y DBA). |

## Metodología de Evaluación
Cada nivel finaliza con un proyecto capstone. Estas rúbricas detallan:
- **Funcionalidad**: ¿El código hace lo que se pide?
- **Calidad**: Limpieza, formato, uso de alias.
- **Performance**: Eficiencia (especialmente en nivel Senior).
- **Documentación**: Explicación de decisiones.
